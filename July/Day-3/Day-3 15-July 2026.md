
Imagine Catalyx wants to build a satellite.

A normal software project has:

```
Frontend
Backend
Database
```

A satellite has:

```
Software
Electronics
Battery
Sensors
Communication System
Thermal System
Mechanical Structure
Ground Station
```

Many teams work together.

Now suppose someone writes:

`"The satellite shall transmit data every 10 seconds."`

This requirement is stored in a PDF.

Months later:

- Software team interprets it one way.
- Electronics team another way.
- Testing team another way.

Chaos.
